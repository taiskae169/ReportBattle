## 2. 배틀리포트

### 배틀리포트 작성 플로우

```
STEP 1. 로스터 확정
    └─ GW 공식 앱 export 텍스트 붙여넣기
    └─ 파싱 → 유닛 목록 구성 (이름 정보만, 이미지는 추후)

STEP 2. 맵 배치
    └─ 미션 선택
    └─ Terrain Layout 선택 (미션별 변형 중 선택)
    └─ Deployment Zone + Terrain Layout 이미지 오버레이
    └─ 완성된 배틀맵 위에 유닛 초기 배치

STEP 3. 턴별 행동 기록
    └─ 드래그로 유닛 이동 체크
    └─ 턴별 행동 텍스트 메모 (선택)

STEP 4. 리포트 작성 / LLM 생성 (선택)
    └─ 수동 입력/편집 (LLM 없이도 가능)
    └─ LLM 연동 시: 기록된 행동 기반으로 서술형 배틀 스토리 자동 생성
```

---

### 2-1. 로스터 입력

| 기능 | 설명 | 우선순위 |
|------|------|---------|
| GW 앱 export 파싱 | GW 공식 앱 export 포맷 텍스트 붙여넣기 지원 | 필수 |
| 유닛 목록 구성 | 파싱 결과로 유닛 이름 목록 생성 | 필수 |
| 유닛 이미지 | 추후 지원 예정 | 추후 |
| 파서 확장 구조 | 다른 포맷(BattleScribe 등) 파서를 모듈로 추가할 수 있는 구조 | 필수 |

> **설계 원칙**: 파서를 인터페이스로 추상화하여 GW 앱 외 포맷은 파서 모듈 추가만으로 확장 가능하도록.

---

### 2-2. 맵 시스템

| 기능 | 설명 | 우선순위 |
|------|------|---------|
| Terrain Layout 선택 | GW 공식 지형 배치 이미지 목록에서 선택 | 필수 |
| Deployment Zone 선택 | 미션별 배치구역 이미지 선택 | 필수 |
| 이미지 오버레이 | 두 이미지를 조합해 완성된 배틀맵 표시 | 필수 |
| 유닛 초기 배치 | 배틀맵 위에 유닛 드래그로 초기 위치 설정 | 필수 |

**이미지 구성:**
- **Terrain Layout**: GW 공식 지형 배치도 (지형 위치, 시야 차단 여부, 목표물 위치)
- **Deployment Zone**: 플레이어별 배치구역 (플레이어 1 / 플레이어 2 영역, 목표물 마커)
- 두 이미지를 오버레이하여 완성된 배틀맵 구성

> **이미지 관리 방식**
> - Phase 1: GW 공식 Terrain Layout / Deployment Zone 이미지를 서비스 DB에 저장하여 제공
> - Phase 2: 유저가 직접 이미지를 업로드하거나 커스텀 레이아웃을 생성할 수 있는 기능 추가

---

### 2-3. 턴별 행동 기록

| 기능 | 설명 | 우선순위 |
|------|------|---------|
| 유닛 이동 기록 | 드래그로 유닛 이동, 턴별 위치 저장 | 필수 |
| 턴별 메모 | 주요 행동 텍스트 메모 (선택) | 선택 |

---

### 2-4. LLM 자동 리포트 생성 ⭐ 핵심 기능

| 기능 | 설명 | 우선순위 |
|------|------|---------|
| 수동 리포트 작성 | LLM 없이 직접 입력/편집 가능 | 필수 |
| LLM 스토리 생성 | 턴별 행동 기록 기반으로 서술형 배틀 스토리 생성 | 필수 |
| LLM 선택 | 유저가 사용할 LLM 모델 선택 | 필수 |
| API 키 등록 | 유저 본인의 API 키 사용 (서비스 제공 없음) | 필수 |
| 스트리밍 출력 | 생성 중인 리포트 실시간 표시 | 필수 |
| 리포트 수정 | 생성된 리포트 직접 편집 가능 | 필수 |
| 재생성 | 같은 데이터로 다시 생성 | 선택 |
| 프롬프트 커스텀 | 고급 유저용 프롬프트 수정 | 선택 |

**지원 LLM (LiteLLM):**
- OpenAI GPT-4o
- Anthropic Claude
- Google Gemini
- Ollama (로컬, 고급 유저)

> **정책**: LLM 기능은 선택 사항. API 키 미등록 시에도 수동 입력/편집으로 리포트 작성 가능.

## 출시 단계

### Phase 1 - MVP
- Google 소셜 로그인
- GW 앱 export 포맷 아미리스트 파싱
- 맵 오버레이 (Terrain Layout + Deployment Zone 이미지 조합)
- 유닛 드래그 이동 기록
- 수동 리포트 작성/편집
- LLM 연동 (유저 API 키, 스트리밍)
- 리포트 목록 / 상세 조회
- 유저 프로필 + 전적

### Phase 2 - 고도화
- Discord 로그인 추가
- 추가 아미리스트 포맷 파서 (BattleScribe 등)
- 유닛 이미지/사진 지원
- 팩션별 필터 / 검색
- 댓글 / 좋아요
- 커스텀 Terrain Layout / Deployment Zone 이미지 업로드
- 커스텀 레이아웃 생성 툴 (직접 배치 방식)
- 프롬프트 커스텀

### Phase 3 - 확장
- 타 게임 지원 (Age of Sigmar, Horus Heresy 등)
- 게임별 독립 지형 배치 툴
- 토너먼트 기록
- 통계 대시보드
